# SSRSParamCleanup
SQL Server Reporting Services (SSRS) LogExecution parameter output cleanup

This page will clean output from the SQL Server Reporting Services LogExecution parameters list.

Purpose:
The output of the parameter column is a series of key/value information. This information can be useful to troubleshoot a report. Having
the parameter output in a format that can be copied into SSMS and executed without manipulating it is useful (especially if passing in 
many key/value pairs)

To use: 
Open release/SSRSParameterClean.html
Copy/paste contents to input text area
Choose output use the button to copy/paste
